Core Innovation

THE TECHNOLOGY

The AI Avatar technology enables real-time, knowledge-grounded conversation through a fully integrated pipeline: document ingestion, semantic retrieval, local LLM inference, neural text-to-speech, and frame-accurate lip-sync — all running on local infrastructure with no cloud dependency.

Technology Stack

🗄️
RAG — Retrieval-Augmented Generation

Documents (PDF, CV, manuals) are ingested, chunked, and embedded into a ChromaDB vector store using Ollama embeddings (nomic-embed-text). At query time, the most relevant context is retrieved and injected into the LLM prompt.

🤖
Local LLM — Ollama / phi3:3.8b

Language generation runs entirely on local hardware via Ollama, using the phi3:3.8b model. The model returns structured JSON with text, facial expression, and animation directives — no API costs, full data privacy.

🔊
Text-to-Speech — Microsoft Edge TTS

The generated response is converted to natural speech using edge-tts (en-US-JennyNeural and other voices). Audio is encoded in base64 and streamed to the frontend in real-time.

👄
Lip-sync — Rhubarb Lip Sync

Audio is analysed phoneme-by-phoneme using Rhubarb. Mouth cue data (viseme timecodes) are sent to the frontend where they drive the avatar's morph targets frame-by-frame via useFrame in React Three Fiber.

🎭
3D Avatar — React Three Fiber + GLTF

The avatar is rendered in WebGL using React Three Fiber and @react-three/drei. The GLTF model supports full skeletal animation, morph targets for facial expressions (smile, angry, surprised…), and Mixamo-compatible animation clips.
